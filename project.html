Toxic Comment Classification Models

  • Developed and compared LSTM and Naive Bayes models to classify toxic comments using Kaggle’s Toxic Comment Classification dataset.
  • Applied data preprocessing techniques, including text cleaning, tokenization, and lemmatization.
  • Achieved an F1-score of 0.8457 with the LSTM model and precision of 0.8902 with Naive Bayes.
  • Provided insights into selecting models based on performance needs, balancing accuracy and efficiency.
