3D Object Generation Using GAN

  • Designed and trained a 3D Generative Adversarial Network (GAN) to synthesize realistic 3D objects represented as voxel grids.
  • Built a 5-layer generator and discriminator architecture using ConvTranspose3D and Conv3D layers to learn complex spatial features.
  • Trained the model using binary cross-entropy loss and the Adam optimizer, ensuring stable convergence and high-quality 3D object outputs.
  • Evaluated model performance by visualizing voxel renderings at various epochs, identifying improvements in object shape fidelity and variation.
  • Used Google Colab for training and visualization, allowing for interactive experimentation and GPU-accelerated learning.
  • Tools & Technologies: Python, PyTorch, Google Colab, Conv3D/ConvTranspose3D, Binary Cross-Entropy, Adam Optimizer, Voxel Grids, Jupyter Notebooks

Toxic Comment Classification Models

  • Built and compared LSTM and Naive Bayes models to detect toxic comments using Kaggle’s Toxic Comment Classification dataset.
  • Performed extensive NLP preprocessing, including text cleaning, tokenization, stopword removal, and lemmatization to prepare input data.
  • Achieved an F1-score of 0.8457 with the LSTM model and a precision of 0.8902 with the Naive Bayes classifier, highlighting trade-offs between deep learning and traditional approaches.
  • Conducted model evaluation and comparison to derive insights into choosing algorithms based on real-world needs such as performance, training time, and resource efficiency.
  • Tools & Technologies: Python, TensorFlow/Keras, Scikit-learn, Pandas, NLTK, Matplotlib, Jupyter Notebooks, Kaggle Dataset
